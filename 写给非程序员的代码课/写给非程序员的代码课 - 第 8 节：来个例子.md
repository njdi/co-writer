# 写给非程序员的代码课 - 第 8 节：来个例子

> AI 时代，能看懂代码就够了。

## 谁应该学习这门课

你听说过Vibe Coding，已经在用、或者打算用 Claude Code、Codex、Cursor 这些工具做点东西，但可能会卡在：

- AI 写出来的代码，看不懂在做什么；
- 程序报错了，丢给 AI 改，改了一处又坏一处；
- 想加个小功能，描述半天 AI 总是做不对；
- 觉得代码有问题，但说不清问题在哪。

这门课就是为你准备的。

## 这门课教什么

### 1. 看懂代码的"形状"

看到一段代码，你能认出哪里在拿数据、哪里在做判断、哪里在循环、哪里在调用现成的功能。

### 2. 看懂代码的"流向"

数据从哪来，经过了什么处理，最后到哪去。这是调试的基础。

### 3. 看懂报错信息

报错不是"程序坏了"，而是程序在告诉你它哪里不对。教你怎么读懂这些信息。

### 4. 怎么准确描述问题

学会"现象是什么"、"我期望的是什么"、"我已经试过什么"。

### 5. 安全和常识

不要暴露自己的隐私信息，不要轻易执行删除命令，什么时候必须自己检查确认后再点回车。

## 来个例子

今年 2026 年，张三 28 岁，过完生日就 29 岁。他朋友比他大 3 岁。两人每天各喝 3 杯咖啡，一天加起来要喝几杯？他俩都成年了吗？都还没到退休年龄吗？今天要么是周末、要么是张三放假——只要满足一个，他就能睡到中午。

**Python 代码：**

```python
LEGAL_AGE = 18              # 法定成年年龄
RETIREMENT_AGE = 65         # 退休年龄

age = 28                    # 张三的年龄
age = age + 1               # 过完生日，张三 29 岁了（变量 + 数字）
friend_age = age + 3        # 朋友的年龄，32 岁（变量 + 数字 → 新变量）
total_age = age + friend_age   # 两人年龄之和，61（变量 + 变量）
years_to_retire = RETIREMENT_AGE - age   # 张三离退休还有 36 年（常量 - 变量）

cups_per_person = 3                          # 每人每天几杯咖啡
people_count = 2                             # 几个人
total_cups = cups_per_person * people_count  # 一天总共几杯，6（变量 * 变量）

is_adult = age >= LEGAL_AGE                   # 张三是不是成年了（比较，结果是布尔）
friend_is_adult = friend_age >= LEGAL_AGE     # 朋友是不是成年了（比较，结果是布尔）
both_adult = is_adult and friend_is_adult     # 两人是不是都成年了（逻辑：与）

is_retired = age >= RETIREMENT_AGE                          # 张三是不是退休了（比较）
friend_is_retired = friend_age >= RETIREMENT_AGE            # 朋友是不是退休了（比较）
both_working = not is_retired and not friend_is_retired     # 两人是不是都还在工作年龄（逻辑：非 + 与）

is_weekend = False                            # 今天是不是周末
is_on_vacation = True                         # 张三今天是不是放假
can_sleep_in = is_weekend or is_on_vacation   # 张三能不能睡到中午（逻辑：或）

print("两人年龄之和：", total_age)               # 输出：两人年龄之和： 61
print("离退休还有：", years_to_retire)           # 输出：离退休还有： 36
print("一天总共喝几杯咖啡：", total_cups)        # 输出：一天总共喝几杯咖啡： 6
print("都成年了吗？", both_adult)                # 输出：都成年了吗？ True
print("都还在工作年龄吗？", both_working)        # 输出：都还在工作年龄吗？ True
print("今天能睡到中午吗？", can_sleep_in)        # 输出：今天能睡到中午吗？ True
```

**JavaScript 代码：**

```javascript
const LEGAL_AGE = 18;              // 法定成年年龄
const RETIREMENT_AGE = 65;         // 退休年龄

let age = 28;                      // 张三的年龄
age = age + 1;                     // 过完生日，张三 29 岁了（变量 + 数字）
let friendAge = age + 3;           // 朋友的年龄，32 岁（变量 + 数字 → 新变量）
let totalAge = age + friendAge;    // 两人年龄之和，61（变量 + 变量）
let yearsToRetire = RETIREMENT_AGE - age;   // 张三离退休还有 36 年（常量 - 变量）

let cupsPerPerson = 3;                          // 每人每天几杯咖啡
let peopleCount = 2;                            // 几个人
let totalCups = cupsPerPerson * peopleCount;    // 一天总共几杯，6（变量 * 变量）

let isAdult = age >= LEGAL_AGE;                  // 张三是不是成年了（比较，结果是布尔）
let friendIsAdult = friendAge >= LEGAL_AGE;      // 朋友是不是成年了（比较，结果是布尔）
let bothAdult = isAdult && friendIsAdult;        // 两人是不是都成年了（逻辑：与）

let isRetired = age >= RETIREMENT_AGE;                          // 张三是不是退休了（比较）
let friendIsRetired = friendAge >= RETIREMENT_AGE;              // 朋友是不是退休了（比较）
let bothWorking = !isRetired && !friendIsRetired;               // 两人是不是都还在工作年龄（逻辑：非 + 与）

let isWeekend = false;                            // 今天是不是周末
let isOnVacation = true;                          // 张三今天是不是放假
let canSleepIn = isWeekend || isOnVacation;       // 张三能不能睡到中午（逻辑：或）

console.log("两人年龄之和：", totalAge);               // 输出：两人年龄之和： 61
console.log("离退休还有：", yearsToRetire);            // 输出：离退休还有： 36
console.log("一天总共喝几杯咖啡：", totalCups);        // 输出：一天总共喝几杯咖啡： 6
console.log("都成年了吗？", bothAdult);                // 输出：都成年了吗？ true
console.log("都还在工作年龄吗？", bothWorking);        // 输出：都还在工作年龄吗？ true
console.log("今天能睡到中午吗？", canSleepIn);         // 输出：今天能睡到中午吗？ true
```
