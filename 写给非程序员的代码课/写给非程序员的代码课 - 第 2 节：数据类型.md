# 写给非程序员的代码课 - 第 2 节：数据类型

> AI 时代，能看懂代码就够了。

## 谁应该学习这门课

你听说过Vibe Coding，已经在用、或者打算用 Claude Code、Codex、Cursor 这些工具做点东西，但可能会卡在：

- AI 写出来的代码，看不懂在做什么；
- 程序报错了，丢给 AI 改，改了一处又坏一处；
- 想加个小功能，描述半天 AI 总是做不对；
- 觉得代码有问题，但说不清问题在哪。

这门课就是为你准备的。

## 这门课教什么

### 1. 看懂代码的"形状"

看到一段代码，你能认出哪里在拿数据、哪里在做判断、哪里在循环、哪里在调用现成的功能。

### 2. 看懂代码的"流向"

数据从哪来，经过了什么处理，最后到哪去。这是调试的基础。

### 3. 看懂报错信息

报错不是"程序坏了"，而是程序在告诉你它哪里不对。教你怎么读懂这些信息。

### 4. 怎么准确描述问题

学会"现象是什么"、"我期望的是什么"、"我已经试过什么"。

### 5. 安全和常识

不要暴露自己的隐私信息，不要轻易执行删除命令，什么时候必须自己检查确认后再点回车。

## 数字

代码里的**数字**就是我们平时理解的数字——年龄、身高、价格、温度。

Python：

```python
age = 28
height = 1.75
price = 9.9
temperature = -3
```

JavaScript：

```javascript
let age = 28;
let height = 1.75;
let price = 9.9;
let temperature = -3;
```

和数学课上一样，数字可以是整的（`28`），可以带小数点（`1.75`、`9.9`），也可以是负的（`-3`）。

### 整数和小数

数学里我们会把 `28` 叫整数、`1.75` 叫小数。代码里也是这么分的，只是叫法稍微不一样：

- **整数**（integer）：`28`、`-3`、`0`
- **小数**（float，浮点数）：`1.75`、`9.9`、`-0.5`

Python 严格区分这两种：`3` 是整数，`3.0` 是小数，虽然数学上相等，但在代码里是两种类型。

JavaScript 比较随意：整数和小数都叫 `number`，不区分。`3` 和 `3.0` 在 JS 里就是同一个东西。

> 这个区别平时不太影响阅读代码，知道有这么回事就行。

## 字符串

代码里的**字符串**就是一段文字——名字、城市、一句话、一段地址，都是字符串。

它的标志是**引号**：被引号包起来的内容，就是字符串。

Python：

```python
name = "张三"
city = "杭州"
greeting = "你好，今天天气不错"
```

JavaScript：

```javascript
let name = "张三";
let city = "杭州";
let greeting = "你好，今天天气不错";
```

引号是字符串的"边界"——引号里面的内容会被当成一段文字，而不是代码。

### 单引号还是双引号

Python 和 JavaScript 都允许用单引号或双引号，两种写法效果一样：

Python：

```python
name = "张三"
name = '张三'
```

JavaScript：

```javascript
let name = "张三";
let name = '张三';
```

挑一种顺眼的用就行。一般的习惯是同一份代码里保持统一，不要一会儿单引号一会儿双引号。

### 加了引号，就不是数字了

这是新手最容易踩的坑：

Python：

```python
age = 28
age = "28"
```

JavaScript：

```javascript
let age = 28;
let age = "28";
```

两行看着差不多，但**第一行的 `28` 是数字，第二行的 `"28"` 是字符串**——只是一段恰好长得像数字的文字。

代码里把这两种东西搞混，就会出现各种奇怪的报错或者结果不对。**判断一个值是数字还是字符串，就看它身上有没有引号。** 这是看代码时最基本的一个区分。

## 布尔

现实里有很多信息只有两种答案：

- 是否成年？是 / 否
- 是否登录？是 / 否
- 快递到了没？到了 / 没到

这种**只有"是"和"不是"两种可能**的信息，在代码里就用**布尔**来表示。

布尔只有两个值：**真**和**假**。

Python：

```python
is_adult = True
is_logged_in = False
```

JavaScript：

```javascript
let isAdult = true;
let isLoggedIn = false;
```

### 大小写要注意

布尔的写法在 Python 和 JavaScript 里有个**容易写错的小区别**：

- Python：`True` / `False`，**首字母大写**
- JavaScript：`true` / `false`，**全部小写**

写反了程序就不认了。Python 里写成 `true` 会报错说"不认识这个名字"，JavaScript 里写成 `True` 也是同样的下场。
