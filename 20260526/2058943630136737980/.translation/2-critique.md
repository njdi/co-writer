# 直译稿问题清单

1. **标题啰嗦**："教会我，作为 CEO 该如何管理 ADHD"读起来拗口，断句不自然。可改得更顺，比如"一个身价 3 亿美元的创始人，教我怎么带着 ADHD 当 CEO"。

2. **"一个'陪伴者'"翻译生硬**：body double 直译成"陪伴者"加引号，读者一头雾水。应该用更口语、更说明性的说法，比如"找个人陪着"。

3. **"对他所写内容最有功能性的人"是英语腔**："most functional to what he's writing"直译过来完全不像中文。原意是"挑一个最懂他正在写的东西的人"。

4. **"在漫长的一天之后，很容易错过并专注于评审工作"语序混乱**：原文意思是评审工作很容易被拖到一天结束、人已经很累的时候才做。直译把意思弄反/弄乱了。

5. **"执行功能"偏术语**：executive function 是心理学术语，普通读者不一定懂。可以保留但稍作解释，或用"自控力／把事情张罗起来的能力"这类更白话的说法平衡。

6. **"这个清单在一天有机会之前就做出了决定"太直译**："The list decides before the day has a chance to"很妙，但直译生硬。意思是：清单替你先定好了重点，免得一天里冒出来的杂事抢走注意力。

7. **"系统比意志力更重要"可以更口语**：意思对，但可以更顺，比如"靠的不是意志力，而是一套机制"。

8. **整体语气**：原文是朋友间聊天、分享心得的口吻，直译稿偏书面，需要往轻松、自然的方向调。
