# 到头来，还是网络效应说了算

**作者：** Michael Mignano ([@mignano](https://x.com/mignano))  
**日期：** 2026年5月9日  
**来源：** [It Always Comes Back to Network Effects](https://x.com/Zephyr_hg/status/2053108729504043201)

靠数据"护城河"已经守不住了。过去几十年，企业的核心业务数据被锁在各种软件里，迁移成本太高，换一套系统还不如将就用。

但现在，AI agent 可以直接把这些数据一行一行抠出来，搬到别处。这个过程可能慢，但 agent 不在乎时间——只要背后的人愿意持续掏钱，它就能一直跑。

说到底，这类业务系统存的不过是一堆事实数据，而事实是可以搬走的。现在已经有 AI 创业公司在专门干这件事：派 agent 把客户在竞争对手平台上的 CRM 数据整个迁走。这招说实话挺绝的。而且这套打法不会止步于 CRM——聊天机器人的记忆、AI 工具的个性化配置，迟早都会轮到。

网络效应才是解法。这从来就是。想在 agent 横行的世界里守住自己的产品，你得有网络效应。

agent 能把你 CRM 里的每一行数据都复制走，但它没法让你的客户去开新平台发来的邮件。

agent 能把 Instagram 上的帖子、照片、粉丝关系全部抓走。但它没法让你朋友为了看你发的东西，特意去打开另一个 app。

agent 可以蒸馏一个模型。事实上，大模型本身正在变成大路货——各家能力越来越接近，连训练数据都可以复制（开源社区已经反复证明了这一点）。真正带不走的，是那批正在用你产品的用户：他们的每一次使用，都在生成真实的反馈数据，让下一个版本变得更好。

到头来，还是网络效应说了算。
