# SpaceX + Anthropic，这只是开始

**作者：** Farzad ([@farzyness](https://x.com/farzyness))  
**日期：** 2026年5月7日  
**来源：** [SpaceX + Anthropic is a sign of things to come.](https://x.com/Zephyr_hg/status/2052125273949147490)

SpaceX——Elon 的火箭和 AI 公司——与 Claude 的母公司 Anthropic，刚刚完成了一笔堪称史上最疯狂的交易。

全球最大的太空公司，同时坐拥海量算力、同时正以超2万亿美元估值冲击 IPO（史上最高）……今天，它和公认最强的 AI 公司正式绑在了一起。

我在这个领域跟了14年，这是我见过的最炸裂的公告之一。让我拆解一下：为什么这笔交易意义深远，为什么它对 AI 的长期格局有最疯狂的影响，以及为什么它将以大多数人今天还看不清楚的方式重塑世界经济。

## 先说交易本身。

Anthropic 宣布与 SpaceX 达成合作，将大幅扩充算力——新增 300 兆瓦的计算资源。这意味着 Claude 模型的速率限制会大幅提升，用户再也不会在用 Claude 的时候频繁撞到"超额"的墙。

事情的起因是：Anthropic 的模型太火了，火到他们自己都没准备好。需求暴增，计算资源根本跟不上，客户和企业被整得抓狂。而 SpaceX——现在已经合并了 xAI——恰好手头有大量闲置算力。

Elon 发帖说，他上周花了很多时间与 Anthropic 高层深聊，了解他们如何确保 Claude 对人类有益，他留下了深刻印象。

## 于是，合作就这样诞生了。

具体来说，Anthropic 拿到了 Colossus 1 的全部算力——这是位于田纳西州孟菲斯、当初为训练 Grok 而建的数据中心。超过 300 兆瓦的额外算力，22 万张 NVIDIA GPU（主要是 H100、H200 和 GB200），预计一个月内上线。

效果立竿见影：Claude Pro、Max、Code、Team、Enterprise 全线速率限制今天起翻倍，Pro 和 Max 账户的高峰限速取消，Opus 模型的 API 上限提高。

要理解为什么这不只是一次"提速"，你得先搞清楚什么是 AI 超大算力商（Hyperscaler）。

Hyperscaler 是个现代词，指的是那一小撮真正有能力大规模训练和运行前沿 AI 模型的公司——他们得同时拥有算力、芯片、电力和物理基础设施。微软 Azure + OpenAI 是一个，Google Cloud + Gemini 是一个，亚马逊 AWS + Anthropic 租户模式是一个，CoreWeave、Oracle、Meta 各有自己的内部技术栈。

过去这些年，从 GPT-3 的时代开始，默认的格局是：微软、谷歌、亚马逊、Meta 这些大科技公司，拥有电力和数据中心；AI 实验室是坐在这套基础设施上的租户。

## 模型是产品，算力是房东。这是截至今天之前的世界法则。

要理解为什么这一次不一样，我们要拼三块关键的拼图。

## 拼图一：宿敌变盟友。

两个月前，Elon Musk 还在公开骂 Anthropic"反人类"、"邪恶"，嘲笑 Anthropic CEO Dario Amodei 写的关于 AI 意识的文章。Elon 和 Dario 之间，没有一点余地。

有意思的是，他们都离开过 OpenAI——Musk 在2018年，Amodei 在2021年，离开的理由都指向了对 Sam Altman 的不信任。而且他们都去建了 Sam Altman 公司的竞争对手。

但 Elon 和 Dario 本人，在公开场合是敌人。现在，至少他们是合作伙伴。也许是那种"把敌人留在身边"的策略。我们大概永远不会真正知道。

## 拼图二：SpaceX 吃掉了 xAI。

2026年2月2日，SpaceX 以全股票形式吸收了 xAI，估值约1.25万亿美元——按今天的估计，已经接近2万亿。路透社称这是人类有史以来最大的企业并购。

xAI 是 Elon 的 AI 实验室，Grok 的制造者。而 Colossus——跨越两座建筑、约55.5万张 GPU 的孟菲斯巨型数据中心——在那笔交易里成了 SpaceX 的资产。SpaceX 已经不只是一家火箭公司，它同时是一家 AI 实验室，而且它拥有支撑这个实验室运转的全部算力。

## 拼图三：Anthropic 被五角大楼踢出局。

今年早些时候，五角大楼与 SpaceX、OpenAI、谷歌、微软、NVIDIA、AWS、Oracle 以及一个叫 Reflection 的小玩家达成了 AI 合作协议——而 Anthropic 被明确排除在外。为什么？因为大约一个月前 Anthropic 与国防部之间那场公开的撕逼。这很重要，我稍后会解释。

与此同时，Anthropic 还面临着另一个越来越公开的问题：算力严重告急。2026年上半年，Claude Code 用户——这是 Anthropic 对标 Cursor、Codex 的开发者产品——几个小时就能把速率限制耗尽，而不是几天。一篇又一篇文章指向同一个结论：Anthropic 被需求掐住了脖子。他们在某些维度上有最好的产品，但就是没办法快到插上足够多的兆瓦，算力跟不上。

## 客户已经被搞崩了。

Anthropic，在很多人眼中拥有地球上最强的模型，却不得不亲手限制自己的用户。

这家公司的估值在本次录制时大约是1.2万亿美元——荒谬的数字。收入在暴力攀升：2025年底年化收入约90亿美元，到3月达到约190亿，到4月第一周已经是300亿。现在可能已经接近400亿了。

从90亿到300亿，大约用了100天——这是人类商业史上任何公司最快的收入增速。

这就是这盘局：Anthropic，产能告急、被五角大楼拒之门外，却拥有软件史上最陡峭的收入曲线。SpaceX，刚吃下 xAI，坐守一座城市大小的孟菲斯数据中心，手里有个叫 Grok 的模型，坦白说，商业竞争力不行。而这两者中间，还横着 Elon 和 Dario 那场肥皂剧式的个人恩怨。

这就是背景。现在说说真正的影响。

## 第一重影响：瓶颈打开了。

Anthropic 增长的算力瓶颈，至少暂时解除了。Anthropic 拿到了更多产能，SpaceX 拿到了一个付费的前沿模型租户带来的大规模稳定收入。两个前敌人意识到，他们共同的问题——OpenAI——比彼此之间的个人恩怨更大。

而且，Anthropic 现在有足够的跑道来发布下一个模型：Mythos。据说这个模型强到"会毁灭世界"。

我们都知道这是夸张。但基准测试不会说谎，这个模型看起来极其强大。现在 Anthropic 能把它真正送到用户面前，进一步拉开领先。

## 第二重影响：SpaceX 已经不是一家火箭公司了。

这是大多数人还没看到的事情：SpaceX 已经悄悄地，一层一层，建起了地球上唯一一套完全垂直整合的 AI 基础设施栈。

**发射层。**

SpaceX 有猎鹰9号，发射成本正在以数量级下降。但更重要的是星舰——当它真正稳定飞起来，目标是每公斤进轨费用10到100美元，比现在猎鹰9号的价格便宜30到300倍，比十年前行业水平便宜超过一千倍。未来，星舰会把数据中心直接送上太空，用太阳的无限免费能源驱动超级智能 AI。

**芯片层。**

SpaceX 和特斯拉正在奥斯丁建造 Terafab，一个2纳米芯片工厂。最初公告是250亿美元的项目，现在预算已经奔着2000亿走了。

就在今天 Anthropic 公告前三小时，Kobeissi Letter 报道 SpaceX 正在为第一期提议550亿美元资本支出，全阶段可达1190亿，英特尔今年4月也加入了作为制造合作伙伴。路线图：每月10万片晶圆，最终扩到100万片——大约相当于台积电全球产能的70%。

**算力层。**

Colossus 1：22万张 GPU。Colossus 2：千兆瓦级设施，另有33.5万张 GPU。密西西比州 Southaven 的第三个站点正在上线，2026年底的目标是100万张 GPU。仅孟菲斯综合体就已经投入约180亿美元。

**网络层。**

Starlink：超过1万颗在轨卫星，人类历史上规模最大的商业卫星星座，未来几年目标是100万颗。

**模型层。**

SpaceX 不仅有 xAI 和 Grok。今天开始，他们还在向 Anthropic 出租前沿模型算力——Anthropic 和 Grok 成了同一个数据中心的共同租户。而在开发者界面这一层，SpaceX 持有600亿美元的期权，可在2026年底前收购 Cursor，毁约金100亿美元。顺带一提，Cursor 是目前最领先的 AI 原生开发工具，底层跑的正是 Claude。

把这些叠起来：发射、芯片、算力、卫星、模型、开发者界面——

以 SpaceX 为中心的马斯克工业栈，拥有六层。其他玩家最多凑出四层，能到四层的已经算幸运。

## 第三重影响：赢家和输家。

**Anthropic 赢了，但有代价。**

产能告急的问题，至少暂时解了。Claude Code 用户立刻得到更多额度。随着产能利用率提升，每个 token 的算力成本下降，300亿美元年化收入的利润结构会改善。下一个模型 Mythos——4月向有限合作伙伴预览，100万 token 上下文窗口，据说是有史以来测试过的最强模型——现在可以服务更大的用户规模了。

但代价是真实的：每年可能要向 xAI 支付数十亿美元的租金。他们现在在结构上绑定了一个正在帮五角大楼建军事 AI 的承包商，而他们自己恰恰被五角大楼列入了黑名单。讽刺。

**SpaceX 赢得最彻底。**

他们刚刚公开证明了自己作为 AI 基础设施公司的核心逻辑：拥有一个付费的前沿模型租户。这跟微软对 OpenAI 的模式一模一样——正是这个模式支撑了微软 Azure 超过1万亿美元的估值。

SpaceX 的 IPO 目标是2026年中。美国银行预测公开上市时估值1.5到2万亿美元起跳。有了这笔交易，那个数学更好算了：稳定的 AI 算力租金收入 + 发射收入 + Starlink + Cursor 期权 + Terafab 股权 + 潜在的与特斯拉合并。简直是不可思议的位置。

**xAI 和 Grok，输了一局。**

xAI 建了 Colossus 1 来训练 Grok。SpaceX 收购了 xAI。今天，SpaceX 把 Colossus 1 的每一兆瓦都租给了 Anthropic。Grok 的训练管道被挪到 Colossus 2 和密西西比站点——xAI 没垮，还有 Colossus 2、密西西比100万 GPU 的路线图，还有 Cursor 期权。

所以我不会说 Grok 死了。但我要说的是：2026年，几乎所有原始 xAI 联合创始人（除 Elon 以外）都已离开，基本上是整个创始团队。今天这笔交易的潜台词已经很清楚——SpaceX 觉得 Anthropic 的美元比 Grok 的市场地位更值钱。Grok 的处境看起来很脆弱，能不能重新爬回前沿模型的位置是个真正的开放问题，当然也没有人应该把 Elon 排除在任何可能性之外。但截至今天，出租算力赚的钱，远比 Grok 跑出来的那点收入值钱。

## 第四重影响：整个系统的逻辑。

好好看这个。

AI 需要能源，能源需要千兆瓦，千兆瓦需要土地、审批和时间——建一个认真的数据中心，最快也要24到36个月。当然，Elon 能4个月搞定。

AI 需要芯片，前沿芯片需要工厂，工厂从零到量产至少需要三四年——而整个链条卡在 ASML，ASML 造的是造芯片的机器。

AI 需要网络来传数据，需要开发者界面来触达人类，开发者界面需要装机基础。最后是模型——所有人都盯着模型，因为模型是能看见的那一层，但模型其实是现在最容易被商品化的部分。来自中国、Mistral、Meta 的开源发布，正在把"最强"和"够用"之间的差距从年压缩到月。

## 护城河不再在模型。护城河在模型之下的一切。

谁能拿下发射、芯片、工厂、千兆瓦、卫星、开发者界面、模型这整条链，谁就能从链条上所有人那里抽利润——包括曾经并肩的前沿实验室。

拿不下这些层的实验室，比如 OpenAI，在结构上就成了依赖者。微软拥有 OpenAI 的算力。谷歌部分拥有 Anthropic 的算力。亚马逊部分拥有 Anthropic 的算力。而从今天开始，SpaceX 拥有 Anthropic 部分算力，拥有 Grok 的算力，持有 Cursor 的收购期权，在与特斯拉合作建芯片，在把日益将托管数据中心的卫星打上天。

太野了。

过去十年，大家都以为 AI 的赢家是那家有最聪明研究员和最好基准分数的公司。

## 那个时代结束了。

今天，AI 的赢家是那家拥有最多物理层的公司。

算力是新石油。

兆瓦是新管道，芯片是新炼油厂，发射是新货轮，开发者界面是新加油站，模型是新汽油——而汽油正在商品化，所以汽油其实是最便宜的那部分。因此，拥有链条其余环节的那家，赢——而且赢得很彻底。

说到这里，你也许觉得最显眼的解读是：Dario 和 Elon 化干戈为玉帛，两个人联手对付 Sam Altman，OpenAI 麻烦大了——尤其是现在 Elon 和 Sam 还在打官司。

两个前 OpenAI 同事联合起来打第三个，这有一定的道理。OpenAI 确实面临真实的困境。他们的 CFO 最近封顶了微软的收入分成。他们从 Stargate 挪威数据中心撤退了，微软通过一家叫 Nscale 的公司自己接手，合同签了3万张 Rubin GPU。

OpenAI 向微软 Azure 承诺了六年2500亿美元，但微软2026年1900亿美元的资本支出、AI 基础设施两年翻倍，已经说明了谁在这段关系里更强势。

是的，OpenAI 被夹住了。

但这里有个现实：两个月前，SpaceX 用约2500亿美元股票吞下 xAI，目的就是进入 AI 竞赛，让 Grok 正面硬刚 Claude、ChatGPT 和 Gemini。

今天，这套逻辑已经变了。Grok 现在是那个相对闲置的内部产品。SpaceX 在 AI 领域真正的商业模式，不是赢得模型竞赛。

## 而是当所有人的房东。

他们不需要打败 Claude。他们只需要 Claude，以及之后的每一个模型，住在他们的算力上、他们的芯片上、他们的电力上、他们的卫星上——尤其是他们正在疯狂扩张星舰产能和 Terafab 产能的当下。SpaceX 即将建造的芯片和火箭数量，会荒谬到一个新的量级。

这正是特斯拉当年对传统车厂做的事，只不过现在应用到了 AI。

想想看：特斯拉垂直整合了供应链，自建电池、自研 FSD 芯片、自己的软件栈、自己的工厂。等到通用、福特、大众开始追，底层的整个工业逻辑已经变了。

传统车厂输掉电动车竞赛，是因为在自动驾驶电动车的世界里，他们不拥有车子之下的那些层。SpaceX 在 AI 领域做的是完全相同的布局：拿下模型之下的一切，这样无论谁赢了模型竞赛，都要向他们交租。永远。

AI 竞赛里的玩家更强也更多，竞争会持续几十年，但现实是：SpaceX 现在是地球上唯一完全垂直整合的 AI 超大算力商，这意味着在本十年剩余的时间里，SpaceX 最有能力拿下 AI 经济中最大的那一块蛋糕。

## 这件事，影响到几乎每一个人。

如果你做软件，你的工具即将变得更强大，而那些把工具送到你手里的实验室，越来越将成为 SpaceX 基础设施上的租户。

如果你做任何知识工作，那些会取代或增强你工作流的 AI 助手，越来越将跑在 SpaceX 的算力上。

如果你是美股七巨头（Mag 7）的投资者——微软、谷歌、亚马逊、NVIDIA、Meta、苹果、特斯拉——你得更新你的模型了。这些名字背后，隐含了一个四方竞争 AI 基础设施的假设。现在是五方。第五方，SpaceX，是唯一一个把发射和芯片制造放在同一屋檐下的。

2030年一个周二的早晨，当一家公司拥有火箭、拥有芯片工厂、拥有千兆瓦电力、拥有卫星、拥有宽带、拥有 AI 的开发者入口，并向地球上其他所有公司出租最聪明的大脑——那个世界，究竟是什么样子？

仔细想想，真的令人头皮发麻。
