# 如何制作令人惊叹的 HTML 幻灯片（完全初学者指南）

**作者：** Zara Zhang ([@zarazhangrui](https://x.com/zarazhangrui))  
**日期：** 2026年5月12日  
**来源：** [How to make stunning HTML slides (for complete beginners)](https://x.com/Zephyr_hg/status/2053953371762286954)

不要做 PowerPoint。做 HTML 幻灯片。

如果你想采用一种更 AI 原生的工作方式，却不知道从哪里开始，那你能做的最简单、回报最高的一件事，可能就是开始用 HTML 幻灯片替代所有的 PPT。

HTML 幻灯片现在非常火，原因如下：

- AI 非常擅长生成漂亮的 HTML，尤其是将文本和图像排列成网格式布局（因为训练数据中包含大量网站内容）
- 人类是视觉动物，天生喜欢美好的东西。幻灯片最重要的一件事，就是抓住观众的注意力
- 制作极快。如果半小时后就要演示，需要一份临时演示文稿，你根本没时间在 PowerPoint 里反复折腾。只需把内容交给 AI，几分钟内它就能为你生成一个漂亮的排版布局
- 用户反馈，观众的反应非常热烈——他们通常从未见过这样的演示，也十分欣赏演讲者对 AI 的熟练运用

以下是针对完全初学者的实用指南，不预设任何编码、使用 agent 或了解 GitHub 等方面的基础知识：

1. 前往 AnyGen（可以免费开始并获得免费积分）
2. 选择"Build slides" -> "Frontend"，然后选择你喜欢的模板
3. 在提示框中输入创建幻灯片的指令
4. 迭代和编辑。所有 HTML 演示文稿在 AnyGen 中都可以编辑！只需点击任何文字进行编辑，或点击"Boss mode"进行批量定向编辑：只需对元素添加注释，告诉 AI 要更改什么

如果你想插入自己的图片，只需将它们发送给 AI 并告诉它在哪里插入

![](images/img-01.jpg)

5. 所有 HTML 演示文稿都可以作为链接分享，直接发给别人即可（无需学习如何部署网站）
6. 所有 HTML 演示文稿都可以导出为 PDF 或 PPT 文件

![](images/img-02.jpg)

你的下一份演示文稿不需要是 PowerPoint。HTML 是未来，而开始制作 HTML 最简单的方式，就是从幻灯片开始。
