# 如何制作令人惊叹的 HTML 幻灯片（完全初学者指南）

**作者：** Zara Zhang ([@zarazhangrui](https://x.com/zarazhangrui))  
**日期：** 2026年5月12日  
**来源：** [How to make stunning HTML slides (for complete beginners)](https://x.com/Zephyr_hg/status/2053953371762286954)

不要做 PowerPoint。做 HTML 幻灯片。

如果你想采用一种更 AI 原生的工作方式，但不知道从哪里开始，你能做的最简单、效益最高的事情，可能就是开始用 HTML 幻灯片替代所有的 PPT。

HTML 幻灯片正在席卷全球，原因是：

- AI 非常擅长生成漂亮的 HTML，尤其是将文本和图像排列成盒式布局（因为它们的训练数据包含大量网站）
- 人类是视觉动物。我们就是喜欢美好的东西。你的幻灯片最重要的事情就是吸引某人的注意力
- 速度很快。如果你半小时后有演示，需要一份快速的演示文稿，你根本没有时间在 PowerPoint 里推来推去。只需将内容发送给 AI，它就会在几分钟内神奇地将所有内容排列成漂亮的布局
- 用户告诉我，他们从观众那里得到了极其正面的反馈，因为观众通常从未见过这样的东西，并且钦佩演讲者多么 AI 原生

以下是针对完全初学者的实用指南，不假设你有任何编码、使用 agent 或了解 GitHub 或技能方面的基础知识：

1. 前往 AnyGen（你可以免费开始并获得免费积分）
2. 选择"Build slides" -> "Frontend"，然后选择你喜欢的模板
3. 在提示框中输入创建幻灯片的指令
4. 迭代和编辑。所有 HTML 演示文稿在 AnyGen 中都可以编辑！只需点击任何文字进行编辑，或点击"Boss mode"进行批量定向编辑：只需对元素添加注释，告诉 AI 要更改什么

如果你想插入自己的图片，只需将它们发送给 AI 并告诉它在哪里插入

![](images/img-01.jpg)

5. 所有 HTML 演示文稿都可以作为链接分享，所以你可以直接发给别人（不需要学习如何部署网站）
6. 所有 HTML 演示文稿都可以导出为 PDF 或 PPT 文件

![](images/img-02.jpg)

你的下一份演示文稿不需要是 PowerPoint。HTML 是未来，而开始制作 HTML 的最简单方法就是制作幻灯片。
